## Rodar local

**1. Clone o repositório**

```bash
  git clone https://github.com/joseallef/Jose_Allef_Portfolio
```

**2. Entre na pasta**

```bash
  cd Jose_Allef_Portfolio
```

**3. Instale as dependências**

```bash
  npm install
```

**4. Rodar local**

```bash
  npm run dev
```

**4. O servidor iniciará na porta 3000 - acesse http://localhost:3000**

```bash
  http://localhost:3000
```
